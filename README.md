# Spatial-Frequency Cued GFANC
This repository will host the official open-source implementation of our work:
**Spatial-Frequency Cued Generative Fixed-Filter Active Noise Control Based on Deep Learning in Reverberant Environments** *(submitted to Signal Processing Journal)*.

The code, together with accompanying scripts and documentation, will be released after the paper’s review process is completed.
